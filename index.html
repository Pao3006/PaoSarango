Semana 02

(03/09/22)

¡En esta segunda clase del curso tuvimos una introducción sobre la creación de proyectos y lo que es una bitácora
digital!

Diseño web consta de 4 puntos:

  1. Panificación
  2. Diseño
  3. Implementación
  4. Mantenimiento

Los sitios web tienen 2 lados:

  • Front – end: lo que el usuario ve cuando navega por internet y llega a dar con el sitio web
  • Back – end: todo lo que no se ve, lo que hace funcionar al sitio web

Las paginas consisten en:

  1. Html (estructura)
  2. CSS (presentación)
  3. JS (programación)

El código HTML se puede escribir en cualquier editor de texto plano como el block de notas, pero se trabajará con
editores de códigos como Visual Studio Code.
